# Resume Bullets

- Built ThreatBrief AI, a local Streamlit application that uses Ollama to summarize Wazuh, AWS CloudTrail, GuardDuty, and Prowler security alerts without paid API keys.
- Implemented JSON alert ingestion through paste and file upload workflows with validation, source detection, pretty-printing, and analyst-style Markdown report export.
- Designed SOC triage prompts that produce incident summaries, severity ratings, key evidence, MITRE ATT&CK mappings, false-positive analysis, and response recommendations.
- Created sample security alerts and documentation to demonstrate defensive detection use cases across endpoint, cloud audit, cloud threat detection, and posture-management findings.
